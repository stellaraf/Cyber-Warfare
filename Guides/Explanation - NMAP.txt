Nmap (short for Network Mapper) is a powerful and widely used open-source tool designed for network discovery and security auditing. Developed by Gordon "Fyodor" Lyon, it's a staple in the toolkit of network administrators, system engineers, penetration testers, and security professionals.

âš™ï¸ Core Functions
Nmap offers a broad range of features, including:

1. Host Discovery
Determine which devices are online. This is often done using ping sweeps or ARP requests.
nmap -sn 192.168.1.0/24

2. Port Scanning
Check for open, closed, or filtered ports on target hosts.
nmap -p 1-65535 192.168.1.100

3. Service and Version Detection
Identify services (like HTTP, SSH) and their versions running on open ports.
nmap -sV 192.168.1.100

4. Operating System Detection
Guess the operating system based on TCP/IP fingerprinting.
nmap -O 192.168.1.100

5. Scripting Engine (NSE)
Run custom or predefined scripts to detect vulnerabilities, backdoors, misconfigurations, and more.
nmap --script=vuln 192.168.1.100

6. Traceroute
Map the path packets take to reach a host.
nmap --traceroute 192.168.1.100
